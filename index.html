About Me

I started my web development journey with HTML and CSS, creating responsive static sites on freelance. In February 2017, my first steps in JavaScript were made. Then I changed job and a new position brought me broad experience in Angular, React, Redux, Typescript and Stylus. Working together with backend developers and designers we released 10 web applications using React and Angular. In my spare time I improve my skills by developing my own projects that helps me reach a deeper understanding of technologies. I enjoy creating something new that not only benefits but also has a beautiful and user-friendly interface.
