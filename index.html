About Me

I started my web developer's way with HTML and CSS, for some time I created responsive static sites on freelance. From February 2017 I started learning JavaScript. At the first work on a front-end developer position, I participated in the creation of an ERP system written on company's own framework, at the same time I began to learn UI frameworks like React. Then I changed job and had development experience on Angular, React, Redux, Typescript and Stylus. I developed about ten web applications on React and Angular in a team with backend developers and designers. In my free time I try to improve my front-end development skills (I write my own projects sometimes), and I'm also interested in machine learning and game dev. I like to create something new, with a beautiful and user-friendly interface that benefits.
